Introduction

CSS (Cascading Style Sheets) is the language used to describe the visual presentation of a web page. It controls how HTML elements appear on screen, on paper, or in other media. While HTML is used to structure content, CSS is used to style and layout that content—changing colors, fonts, spacing, alignment, and much more.

CSS Syntax

A CSS rule consists of a selector and a declaration block:

css

selector {
  protery: value;
}

Example:

css

p {
  color: blue;
  font-size: 16px;
}

  • p is the selector (targets all <p> elements).
  • color and font-size are properties.
  • blue and 16px are the values assigned to those properties.

Ways to Apply CSS

  1. Inline CSS – added directly into an HTML element using the style attribute:

    html
    
<p style="color: red;">This is red text.</p>

  2. Internal CSS – placed inside a <style> tag in the <head>:

    html
    
<style> 
  body {
    background-color: lightgray;
  }
</style>

  3. External CSS linked to a separate .css file:

    html
    
<link rel="stylesheet" href="styles.css" />

    External CSS is the most common and maintainable approach.

Common CSS Properties
  • color: Changes text color → color: green;
  • background-color: Changes background color → backgroundcolor: yellow;
  • font-size: Changes text size → font-size: 20px;
  • text-align: Aligns text (left, center, right) → text-align: center;
  • margin: Space outside the element → margin: 10px;
  • padding: Space inside the element → padding: 15px;
Selectors

CSS selectors target HTML elements to apply styles.

Common Selectors:

  • * — Universal selector
  • element — Targets by tag name (p, div, etc.)
  • .class — Targets by class attribute
  • #id — Targets by ID attribute
  • element1, element2 — Targets multiple types of elements
  • element element — Descendant selector
  • element > child — Direct child selector

Example:

css

.container > p {
  margin: 10px;
}

Colors and Units

Colors can be defined in multiple ways:

css

color: red;               /* Named color */
color: #ff0000;           /* Hex code */
color: rgb(255, 0, 0);    /* RGB*/
color: hsl(0, 100%, 50%;  /* HSL*/

Units of measurement:

  • px — pixels
  • em — relative to parent font-size
  • rem — relative to root font-size
  • % — percentage
  • vh, vw — viewport height/width

Box Model

Every HTML element is a box made up of:

  • Content: The text or image
  • Padding: Space inside the box
  • Border: The edge around the box
  • Margin: Space outside the box

Example:

css

div {
  margin: 20px;
  border: 2px solid black;
  padding: 10px;
}

Positioning and Layout

CSS provides several ways to layout elements:

  • static: default layout (no special positioning)
  • relative: positioned relative to its normal position
  • absolute: positioned relative to the nearest positioned ancestor
  • fixed: positioned relative to the viewport
  • sticky: toggles between relative and fixed, based on scroll

Example:

css

.box {
  position: absolute;
  top: 50px;
  left: 100px;
}

Flexbox and Grid

Flexbox: A layout system for 1D (row or column) alignment:

css

.container {
  display: flex;
  justify-content: space-between;
  align-item: center;
}

Grid: A powerful 2D layout system:

css

.grid-container {
  display: grid;
  grid-template-columns: 1fr 1fr;
  gap: 10px;
}

Media Queries (Responsive Design)

Media queries help make your site responsive by applying styles depending on the screen size.

css

@media (max-width: 768px) {
  body {
    font-size: 14px;
  }
}

This adjusts the font size for smaller screens like mobile phones.
